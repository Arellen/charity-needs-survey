# 腾讯云函数 SCF 部署指南

## 第一步：登录腾讯云

1. 打开 https://console.cloud.tencent.com/scf
2. 登录（微信扫码即可）
3. 首次使用需要授权 SCF 服务角色，点"同意"即可

---

## 第二步：创建函数

1. 点击「函数服务」→「新建」
2. 选择「从头开始」
3. 填写：
   - **函数名称**：`feishu-relay`
   - **地域**：选「广州」或「上海」
   - **运行环境**：`Python 3.9`
4. 函数代码 → 提交方法选「在线编辑」→ 把 `scf/index.py` 内容粘贴进去
5. 往下滑，找到「高级配置」→ **执行超时时间** 改成 `10 秒`
6. 点击「完成」

---

## 第三步：创建 API 网关触发器

1. 函数创建完成后，点进函数详情页
2. 点击「触发管理」标签页 →「创建触发器」
3. 配置：
   - **触发方式**：`API 网关触发`
   - **API 服务**：选「新建 API 服务」
   - **请求方法**：`ANY`
   - **发布环境**：`发布`
   - **鉴权方法**：`免鉴权`
4. 点击「提交」

---

## 第四步：获取 URL

1. 创建触发器后，列表里会出现一条记录
2. 复制「访问路径」这一栏的 URL，类似：
   ```
   https://service-xxxxx-1234567890.gz.apigw.tencentcs.com/release/feishu-relay
   ```

---

## 第五步：测一下

把 URL 替换到下面命令中，在终端跑一次：

```bash
curl -X POST '你的SCF地址' \
  -H 'Content-Type: application/json' \
  -d '{"msg_type":"text","content":{"text":"🧪 SCF中继测试"}}'
```
