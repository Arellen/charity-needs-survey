# P0-1：问卷结果持久化存储到飞书多维表格

## 一、目标

每次机构提交问卷，除了飞书群即时通知，自动写入飞书多维表格一行。  
后续可在飞书表格中搜索、筛选、对比历史提交，也能导出 Excel。

---

## 二、架构

```
问卷提交 → SCF 云函数 → 飞书群消息（即时通知）
                      → 飞书多维表格（持久化存储，一行一条记录）
```

SCF 云函数里加一个步骤：收到问卷后，调用飞书开放平台 API，向多维表格追加一行。

---

## 三、你需要准备的东西

### 3.1 创建一个飞书应用

> 为什么需要应用？群机器人 Webhook 只能发消息，不能读写表格。读写表格需要飞书应用 + API 权限。

| 步骤 | 操作 |
|---|---|
| 1 | 打开 [open.feishu.cn](https://open.feishu.cn) → 开发者后台 |
| 2 | 创建「企业自建应用」→ 名称填"问卷存档" |
| 3 | 添加应用能力 → 开启 **「多维表格」** 权限 |
| 4 | 权限范围选 **「读取、写入多维表格」** |
| 5 | 安全设置 → 在「重定向 URL」随便填一个（如 `https://localhost`） |
| 6 | 发布应用 → 版本管理中「创建版本」→ 提交审核（管理员自己审批通过） |
| 7 | 获取 **App ID** 和 **App Secret**（在「凭证与基础信息」页） |

## 五、使用效果

提交后：
- 飞书群收到消息（即时通知）—— **已有**
- 多维表格新增一行（持久化存档）—— **新增**
