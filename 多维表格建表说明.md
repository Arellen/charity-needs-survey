# 飞书多维表格建表说明

> 新建一个多维表格，命名「项目需求管理」，然后按下面建两张工作表。

---

## 工作表1：「项目目录」

| 字段名 | 类型 | 选项/说明 |
|---|---|---|
| 项目名称 | 文本 | 例如「公益小程序」 |
| 项目标识 | 文本 | URL参数用的英文名，如 `charity` |
| 对接机构 | 文本 | |
| 联系人 | 文本 | |
| 联系方式 | 文本 | |
| 项目状态 | 单选 | 需求对接中 / 开发中 / 已上线 / 已完成 / 暂停 |
| 问卷链接 | 链接 | 发给机构填的那个 URL |
| 期望上线 | 文本 | |
| 预算范围 | 文本 | |
| 备注 | 文本 | |

---
