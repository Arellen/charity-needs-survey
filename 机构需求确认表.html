以下介绍小程序各功能模块。阅读完毕后，请跳到下方勾选表单。
A 捐款渠道

🏠 项目广场（首页）

捐款人打开小程序后看到项目列表，可按分类筛选（助学/医疗/救灾/环保等）。每个项目显示封面图、已筹金额进度条、剩余天数。

🖼️ 参考效果：类似"腾讯公益"小程序首页的项目卡片列表

📄 项目详情页

展示完整信息：项目背景、受助群体说明、预算明细表（钱花在哪）、执行计划时间线、已捐款人列表（可匿名）、项目进展动态。

🖼️ 参考效果：顶部大图 → 项目介绍 → 预算表 → 底部捐款按钮

💳 在线捐款

选择金额（固定档位 + 自定义）→ 填写留言 → 是否匿名 → 微信支付 → 捐款成功。全程一分钟完成。

🎖️ 电子捐赠证书

捐款成功后自动生成证书，显示姓名（或匿名）、金额、项目名称、日期，支持保存图片和分享到朋友圈/微信群。

🖼️ 参考效果：类似腾讯公益的捐赠证书，可分享朋友圈

👤 捐款人个人中心

查看我的捐款记录、累计捐款总额、证书合集、已关注的项目。

📬 项目进展推送

项目有重要进展时（已筹50%、开始执行、现场新照片），通过微信服务通知推送给捐款人。

📎 场景化捐款入口（进阶）

将捐款入口嵌入到合作商家场景中：在超市扫码付款时弹出"捐1元"选项；和公众号文章联动，文章内直接捐；生成项目专属二维码印在活动海报上。

🖼️ 参考效果：微信支付爱心餐——麦当劳点餐时弹出"捐1元"

🔔 捐款反馈闭环（进阶）

捐款人不仅看到"我捐了多少钱"，还能收到具体反馈——"您于3月捐款的助学项目，已帮助云南昭通3名初中生完成本学期学业"。

B 受助人管理

📝 受助人信息登记

一线人员现场走访时填写：姓名、性别、年龄、身份证号（系统自动脱敏显示）、联系方式、家庭住址、人口、收入情况等。

📸 困难情况描述 + 拍照上传

选择困难类型（因病/因残/因学/孤寡/受灾等），填写具体描述，拍照上传证明材料（低保证、诊断书、残疾证、村委会证明等）。

📊 需求评估

判断每位受助人需要什么（资金/物资/服务/混合）、预估金额或物资清单、紧急程度（紧急/一般/可排队）。

✅ 多级审核

一线录入 → 项目负责人初审 → 机构负责人终审 → 通过进入"待帮扶库"，不通过退回并附原因。

📋 帮扶记录 + 查询导出

每位受助人的完整帮扶历史：哪笔捐款、什么项目、帮扶了什么、何时、效果如何。支持按姓名、地区、类型搜索筛选，可导出Excel表格。

🤲 代操作功能

考虑到部分受助人（如老人）没有智能手机或不会操作，允许一线工作人员代为登记。

C 项目管理

🚀 项目立项 + 审核

创建新项目：填写名称、类型、目标金额、起止日期、受助对象描述、预算明细、执行计划、上传封面图。提交后机构内部审核，通过后上线展示。

📈 筹款进度实时监控

实时看板：已筹金额/目标金额、捐款人次、日趋势图、剩余天数倒计时。

🖼️ 参考效果：数据仪表盘风格，一目了然

💰 资金管理

每笔支出的用途说明、金额、凭证上传。可用余额 = 已筹总额 - 已支出总额，资金流向完全透明。

📢 项目进展发布

发布图文动态（现场照片+文字+资金使用情况），同步推送给关注该项目的捐款人。

🏁 项目结项

提交结项报告（总筹款、总支出、受助人数、执行总结），审核通过后标记"已完成"，生成公开的结项公示页面。

D 进阶可选功能

💚 尊严支付

不直接免除费用，而是让受助人支付象征性金额（如1分钱/1元）。参考微信支付爱心餐：环卫工人付1分钱就能吃一份正餐。每个项目可单独设置是否开启。

🏢 企业配捐

合作企业出配捐资金池（如"我出10万，用户捐1元我配1元，配完为止"）。捐款人看到自己捐50元变成100元，捐赠意愿更强；企业获得品牌曝光和CSR素材。每个项目可单独配置。

🖼️ 效果参考：项目页显示"XX企业配捐进行中，余额还剩68%"；捐款确认页显示"您捐50 + 企业配50 = 项目收到100"

🤝 合作机构管理

除了自己机构的工作人员，还可以开放权限给外部合作单位（社区、村委会、其他公益组织）。每个合作机构只能看到自己录入的受助人数据。

🔄 月捐（定期自动扣款）

捐款人开通月捐后，每月自动从微信扣款固定金额，持续支持项目。

📺 数据大屏

在办公室电视/大屏幕上实时展示累计筹款总额、在帮项目数、本月新增受助人数等，适合接待参访、理事会汇报。
