# 公益小程序 · 功能介绍与选择清单

> 本文档用于帮助贵机构快速了解小程序能做什么、需要准备什么。
> 请通读后，配合《腾讯问卷》逐项勾选您的需求，我们将据此出具开发方案。

---

## 一、这个小程序能帮机构做什么

简单来说，三件事：

1. **让社会公众方便地给项目捐款**（微信打开 → 看项目 → 支付捐款，一分钟完成）
2. **让一线工作人员现场录入受助人信息**（走访拍照 → 填表 → 提交审核）
3. **让管理人员全程掌控项目进度**（筹了多少钱、帮了多少人、钱花在哪，一目了然）

### 模块 D：可选进阶功能

> 以下功能可根据机构实际情况选择，不影响核心流程的正常使用。

#### D1. 尊严支付（受助人象征性付费）

不直接免除全部费用，而是让受助人支付象征性金额（如 1 分钱/1 元）。参考"微信支付爱心餐"：环卫工人只需付 1 分钱就能吃一份正餐。

**为什么这样做？** 让受助人感觉是"低价购买"而非"被施舍"，保护自尊心。

**可以按项目设置**：有些项目适合（如爱心餐），有些不适合（如大病救助），每个项目创建时单独选择是否开启。

- [ ] 需要，按项目可选开启
- [ ] 不需要
- [ ] 不确定，想了解更多

---

#### D2. 企业配捐

合作企业出配捐资金池。比如企业说"我出10万元配捐，用户每捐1元我配1元，配完为止"。捐款人看到自己的10元变成了20元，捐赠意愿更强；企业也获得了品牌曝光和CSR素材。

**同样按项目可选开启**：每个项目可以选择是否绑定配捐池。

**配捐展示效果**：项目页显示"XX企业在配捐中，配捐余额还剩68%"，捐款确认页显示"您捐50元 + XX企业配捐50元 = 项目收到100元"。

- [ ] 需要，按项目可选开启
- [ ] 以后再说（P2期）
- [ ] 不需要
- [ ] 不确定，想了解更多

---

- [ ] 需要
- [ ] 不需要

#### D4. 月捐（定期自动扣款）

捐款人开通月捐后，每月自动从微信扣款固定金额，持续支持项目。

- [ ] 需要（P1期）
- [ ] 以后再说（P2期）
- [ ] 不需要

---

#### D5. 数据大屏

- [ ] 需要
- [ ] 不需要

## 四、您还需要什么？

以下是我们不做的（请确认）：

- 我们不提供善款的金融托管——资金直接进入贵机构微信支付商户号绑定的对公账户，我们只负责小程序的技术开发和维护
- 我们不涉及受助人身份的真伪鉴定——证明材料审核由贵机构人工完成，系统只提供流程支持

如果以上没问题，请继续填写《腾讯问卷》提交您的需求选择。
